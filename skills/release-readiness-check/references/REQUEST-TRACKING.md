## How to Request an Enhancement

1. Open an issue in this repository using the **Skill Enhancement Request** template.
2. The issue will be labelled `skill-enhancement` automatically.
3. The catalog maintainers will triage, implement, and reference the issue in the release notes.
4. Once released, the row above is updated and the issue is closed.

## Labels Used

| Label | Meaning |
|-------|---------|
| `skill-enhancement` | A requested change to an existing skill |
| `skill-bug` | Incorrect or misleading skill behaviour |
| `needs-triage` | Awaiting maintainer review |
| `accepted` | Accepted, scheduled for next release |
| `released` | Shipped in a catalog release |
